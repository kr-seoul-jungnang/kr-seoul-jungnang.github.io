# 중랑구 · 서울특별시

> 중랑구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-jungnang.github.io/**

중랑구는 구를 남북으로 가로지르는 하천 중랑천에서 이름을 따온, 서울 동북권의 대표적인 주거·생활 자치구입니다. 1988년 동대문구에서 갈라져 나온 비교적 젊은 행정구역이지만, 그 안에는 조선시대 봉수가 오르던 봉화산과 서울 옹기의 70% 이상을 빚어내던 신내동 옹기마을, 그리고 한용운·방정환·유관순이 잠든 망우역사문화공원처럼 근현대사의 결이 겹겹이 쌓여 있습니다. 봄이면 중랑천 둔치를 따라 5.45km에 이르는 국내 최대 장미터널이 붉게 물들어 서울장미축제로 300만 명이 넘는 인파를 불러 모읍니다. 용마산 절벽을 깎아 만든 51.4m 인공폭포가 도심 한복판에서 물보라를 뿜고, 7호선 여섯 개 역이 면목·상봉·중화 일대를 촘촘히 잇는 곳이기도 합니다. 화려한 상업 중심지보다는 산자락과 하천, 옹기가마와 묘역의 이야기가 어우러진 생활밀착형 동네가 중랑구의 참모습입니다.
